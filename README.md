- Steps to download the files of your project:
  1. Open Github
  2. Look for your file name
  3. Download the Zip file
  4. Extract the files into your Apache Netbeans
  5. Open Netbeans
  6. Go to file & click open project
  7. Look for the extracted file
  8. Finish
